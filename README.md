# instagram-clone

Created with React, GraphQL, and Firebase! I built this project to further develop my understanding in full stack development.

# Link
https://instagram-nu-clone.vercel.app

## Routes

- / (feed page)

Components:

- FeedPost
- FeedPostSkeleton
- FeedSideSuggestions

- /explore (explore page)

Components:

- ExploreSuggestions
- ExploreGrid

- /p/:postId (post page)

Components:

- Post
- PostSkeleton
- PostModal
- MorePostsFromUser

- notifications

Components:

- NotificationList
- NotificationTooltip

- /:username (profile page)

Components:

- ProfileTabs

- /accounts/edit/ (edit profile page)

- /accounts/login (login page)

- /accounts/emailsignup/ (signup page)

- - (not found page)
